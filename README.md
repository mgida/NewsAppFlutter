# news_app

A Flutter application for searching , retrieving articles and saved them to read later

developed with :
* chopper
* json_serializable
* moor db
* provider

## Before starting 

get your api_key and put it in lib\utils\constants file

- [Get your api_key](https://newsapi.org/)
